Mathematik

Fundierte Kenntnisse in der Mathematik sind notwendig, um die komplexen Vorgänge in realen dynamischen Systemen beschreiben zu können. So können mathematische Modelle von der Wirklichkeit erstellt und analysiert werden. Simulationen sind virtuelle Experimente an diesen mathematischen Modellen. Für Berechnungen werden numerische Algorithmen eingesetzt, deren Verständnis für die Analyse der Simulationsergebnisse wichtig ist.

Das Hookesche Gesetzt

Die Spannung σ beschreibt das Verhältnis einer Kraft F zu einer Fläche A.

σ = F/A

Die einwirkende Kraft bewirkt eine Formänderung. Im Falle eines eindimensionalen Stabes kann diese Formänderung durch eine Länge Δl beschrieben werden. Das Verhältnis zur ursprünglichen Länge des Stabes l0 ist die Dehnung ε.

ε = Δl / l0

Der lineare Zusammenhang zwischen der Spannung σ und der Dehnung ε wird durch das Elastizitätsmodul E beschrieben.

σ = E ∙ ε

Kleines Pseudocode-Programmierer-Glossar

Ein Programmcode wird oft als Pseudocode dokumentiert. Die dabei verwendete Syntax entspricht nicht einer spezifischen Programmiersprache, daher kann der Code so nicht direkt von einer Maschine interpretiert werden. Er dient zur Beschreibung der Programmstruktur unabhängig der verwendeten Technologie und kann als Grundlage zur Implementation in jeder beliebigen Programmiersprache verwendet werden.

Array Eine einfache Datenstruktur, in welcher Daten als eine Liste abgebildet werden. Jeder Listeneintrag enthält einen eindeutigen Index.
if Dadurch kann im Code mittels einer Bedingung eine Verzweigung erzeugt werden. Ein Teil des Codes wird ausgeführt falls die Bedingung wahr ist, der else-Teil wird ausgeführt falls die Bedingung nicht zutrifft.
for Die for-Schleife wird genutzt um Code mehrfach auszuführen. Dabei wird ein Index kontinuierlich hochgezählt.
while Programmcode innerhalb der while-Schleife wird solange die Bedingung wahr ist ausgeführt.
% Der Modulo-Operator beschreibt eine Division mit Rest. So ist z.B. das Ergebnis von 17%6 die Fünf, da 6 ∙ 2 + 5 = 17 ist.
